// ℹ HOW COUPONS WORK

  • Percentage (e.g. 20%) takes a % off the order total.
  • Fixed (e.g. 50 EGP) subtracts a flat amount from the order.
  • Min Order requires the cart total to meet a minimum before the code applies.
  • Max Uses caps how many times the code can be redeemed (left empty = unlimited).
  • Duration — set either Valid for N days (relative) or a specific expiry date. Leave both blank for no expiry.
  • Each successful order atomically increments usedCount. Codes that hit maxUses or pass their expiry are auto-rejected at checkout.
